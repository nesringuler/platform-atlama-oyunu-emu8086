# Platform Atlama Oyunu (EMU8086)

Bu projede 8086 Assembly dili kullanılarak basit bir platform atlama oyunu geliştirilmiştir.

## Özellikler
- Karakter zıplama mekanizması
- Engel sistemi
- Skor sistemi
- Game Over ekranı

## Kullanılan Teknolojiler
- EMU8086
- 8086 Assembly

## Dosyalar
- Platform_Atlama_Oyunu.asm → Oyun kodu
- Platform_Atlama_Oyunu_Rapor.pdf → Proje raporu

## Çalıştırma
EMU8086 programında .asm dosyasını açarak çalıştırabilirsiniz.
